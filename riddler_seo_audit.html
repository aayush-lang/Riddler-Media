Google Search Console
Open GSC → Performance → set the date range. The 4 summary numbers at the top are what you need here.
GSC → Performance → Overview (top 4 tiles)
?
Total ClicksTimes someone clicked your site from Google search results.
GSC → Performance → "Total clicks" tile (top)
?
Total ImpressionsHow many times the site showed up in search results (clicked or not).
GSC → Performance → "Total impressions" tile (top)
?
Average CTR% of people who saw the site in results and clicked it. GSC shows it with a % sign — just enter the number.
GSC → Performance → "Average CTR" tile (top)
?
Average PositionAverage Google ranking across all keywords. 1 = top result. Lower = better.
GSC → Performance → "Average position" tile (top)
?
Highest impression pageThe page appearing most in search results — usually the homepage or a key blog post.
GSC → Performance → Pages tab → sort by Impressions ↓
?
Top search queriesThe actual words people typed to find the site.
GSC → Performance → Queries tab → top 5–10 rows
Step 3 of 5
Technical health
All these numbers live in SEMrush Site Audit. Run a fresh crawl first if it's been more than 2 weeks.
SEMrush → Site Audit → Overview
?
Site Health ScoreSEMrush's overall technical health score. 90%+ is good. Below 80% needs attention.
SEMrush → Site Audit → the big % circle on the overview page
?
Pages crawledHow many pages SEMrush found vs the crawl limit.
SEMrush → Site Audit → "X/100 pages crawled" near the top
?
ErrorsSerious issues that directly hurt rankings — broken links, missing titles, server errors.
SEMrush → Site Audit → red "Errors" count
?
WarningsIssues that hurt rankings but aren't critical — missing metas, duplicate content, slow images.
SEMrush → Site Audit → yellow "Warnings" count
?
Crawlability% of pages Google can actually access and index. Should be 90%+.
SEMrush → Site Audit → "Crawlability" score card
?
HTTPS% of pages served securely. Should be 100%. Anything below means some pages are on HTTP — Google penalises this.
SEMrush → Site Audit → "HTTPS" score card
?
Internal LinkingHow well pages on the site link to each other. Good internal links help Google discover all pages.
SEMrush → Site Audit → "Internal Linking" score card
?
Site PerformanceHow fast pages load. Google uses speed as a ranking factor. 90%+ is ideal.
SEMrush → Site Audit → "Site Performance" score card
?
Core Web VitalsGoogle's page experience scores (LCP, CLS, INP). A direct ranking factor.
GSC → Experience → Core Web Vitals — OR — run PageSpeed Insights on the homepage
Step 4 of 5
Domain authority
How much Google trusts this domain. Type the client's URL into SEMrush Domain Overview — all numbers are on that one page.
SEMrush → Domain Overview → enter domain
?
Authority ScoreSEMrush's domain trust score. New sites = 1–5. Established brands = 40–80+. Directly affects how easily pages rank.
SEMrush → Domain Overview → big number top-left labelled "Authority Score"
?
Referring DomainsNumber of unique websites linking to this site. Quality matters more than quantity.
SEMrush → Domain Overview → "Ref. Domains" number
?
BacklinksTotal links pointing to the site (can include multiple from the same domain).
SEMrush → Domain Overview → "Backlinks" number
?
Organic KeywordsNumber of keywords the site ranks for in Google's top 100. The most direct measure of SEO reach.
SEMrush → Domain Overview → "Organic Keywords" number
?
Organic TrafficSEMrush's estimate of monthly organic visitors. May differ from GA4 — use GA4 for actuals, this for context.
SEMrush → Domain Overview → "Organic Traffic" number
